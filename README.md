# TA_2_PEMWEB
Pengumpulan Tugas Akhir Praktikum Pemrograman Web 



Halo! Ini adalah proyek website portofolio pribadi yang saya buat untuk menampilkan profil diri, informasi akademik, serta minat dan keahlian saya di bidang pengembangan front-end dan desain. Website ini dikembangkan menggunakan HTML dan CSS sebagai bagian dari latihan pengembangan dan media representasi profesional diri saya secara digital.

Rancang Bangun dan Tujuan
Website ini saya rancang untuk memperkenalkan siapa saya, latar belakang pendidikan saya di Teknik Informatika Universitas Lampung, serta ketertarikan saya dalam dunia pengembangan perangkat lunak dan desain antarmuka. Halaman ini memiliki beberapa bagian utama:

- Header yang menampilkan logo "Welcome to My Web" dan navigasi sticky.
- Tentang Saya, berisi ringkasan latar belakang, Biodata Singkat (NPM: 2315061005), serta filosofi hidup ("Jika tidak bisa membahagiakan, setidaknya jangan mengecewakan.").
- Keahlian Saya yang disusun dalam bentuk kartu dan memamerkan kemampuan di bidang HTML5, CSS3, dan Figma.
- Kontak yang mencantumkan alamat email dan nomor telepon.
- Footer sederhana yang menjadi penutup halaman.
